# 🌍 DreamTravel - Agencia de Viajes Premium

Este es un proyecto de **Landing Page** para una agencia de viajes de lujo, diseñado con un enfoque en la experiencia de usuario (UX) y una estética visual atractiva que invita a la exploración.

## 🚀 Características Principales
* **Diseño Visual Premium:** Uso de tipografías elegantes (Playfair Display) y texturas de fondo (mapamundi, papel antiguo).
* **Secciones Dinámicas:** Galería de destinos, tarjetas de paquetes turísticos y prueba social mediante testimonios.
* **Conversión Directa:** Integración de un botón flotante de WhatsApp animado y formulario de contacto.
* **Favicon Personalizado:** Identidad visual desde la pestaña del navegador.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica clara.
* **CSS3:** Layouts con Flexbox y CSS Grid, animaciones personalizadas (pulse, fadeIn) y diseño responsive.
* **Google Fonts:** Tipografía profesional para mejorar la legibilidad.
